# AI SmartBuild Installation Guide

This guide is intended for first-time users of Revit, pyRevit, and Python. Follow the steps in order.

## 1. Install Revit

Target:

- Install `Revit 2024`

Expected result:

- `Autodesk Revit 2024` appears in the Windows Start menu
- Revit launches normally

Common errors:

- Incomplete installer: Re-download the installer and try again
- License issues: Verify that your school or personal account has an available license

## 2. Install pyRevit

Steps:

1. Open the pyRevit releases page: `https://github.com/pyrevitlabs/pyRevit/releases`
2. Download the latest installer
3. Double-click to install using default options

Expected result:

- `pyRevit CLI` appears in the Start menu
- After opening Revit, the `pyRevit` tab appears in the top Ribbon

Common errors:

- pyRevit tab does not appear in Revit: Restart Revit and try again
- Still not working after installation: Uninstall, reinstall, and verify that your Revit version is compatible with pyRevit

## 3. Get the Project Code

Recommended method:

1. Install Git
2. Open Command Prompt or PowerShell
3. Run:

```bash
git clone <your-repo-url> AI-Smart-Build
cd AI-Smart-Build
```

If you are not familiar with Git, you can also:

1. Click `Code` on the repository page
2. Select `Download ZIP`
3. Extract to a location you can easily find, such as the Desktop

Expected result:

- The directory contains `AISmartBuild.extension`
- The directory contains `scripts/install_windows.bat`

Common errors:

- `git` not found: Install Git first, or use the ZIP download method instead
- Chinese characters in deeply nested paths: Place the project in a short English path, e.g., `D:\Demo\AI-Smart-Build`

## 4. Run the Windows Installation Script

Steps:

1. Navigate to the project root directory
2. Double-click `scripts/install_windows.bat`

The script does three things:

- Checks for Python
- Installs `openpyxl`
- Creates the `config.json` configuration file

Expected result:

- The command window shows `=== 安装完成 ===`
- A configuration file appears in your user directory:

```text
C:\Users\<your-username>\.ai-smart-build\config.json
```

Common errors:

- Python not detected: Install Python 3.10 or later and check "Add Python to PATH" during installation
- `pip install openpyxl` fails: Check your network connection, or re-run the terminal with administrator privileges

## 5. Configure the DeepSeek API Key

Steps:

1. Open the configuration file:

```text
C:\Users\<your-username>\.ai-smart-build\config.json
```

2. Replace the `DEEPSEEK_API_KEY` value with your actual key

Example:

```json
{
  "DEEPSEEK_API_KEY": "your-actual-api-key",
  "DEEPSEEK_MODEL": "deepseek-chat",
  "DEEPSEEK_API_URL": "https://api.deepseek.com/v1/chat/completions"
}
```

Expected result:

- The file is saved successfully
- `DEEPSEEK_API_KEY` is no longer a placeholder

Common errors:

- Forgot to save the file: Confirm that the file is saved after editing
- JSON format error: Make sure there is no trailing comma after the last field

## 6. Register the pyRevit Extension Path

Steps:

1. Open the pyRevit settings UI or pyRevit CLI
2. Add "the directory containing `AISmartBuild.extension`" to pyRevit's custom extension path

For example, if the project is at:

```text
D:\Demo\AI-Smart-Build
```

Then the path to add is:

```text
D:\Demo\AI-Smart-Build
```

Not just the `AISmartBuild.extension` directory itself.

Expected result:

- The new custom extension path is visible in pyRevit settings

Common errors:

- Wrong path selected: You must select "the parent directory that contains the extension directory"
- Directory was moved: If you move the project directory later, you need to re-register the path

## 7. Reload and Verify the Plugin

Steps:

1. Open Revit
2. Execute `pyRevit Reload`
3. Check whether the `AI智建` tab appears in the Ribbon
4. Click the `关于` (About) button to confirm that version and environment info is displayed

Expected result:

- `AI智建` appears in the Ribbon
- You can see buttons for one-click generation, AI chat, model export, etc.
- The About dialog displays version, pyRevit, Revit, and configuration status correctly

Common errors:

- Buttons not visible: First check that the extension path is registered correctly, then execute `pyRevit Reload`
- API Key not configured message: Go back to step 5 and check `config.json`
- Button click has no response: Run the environment check script first to verify dependencies and directory structure

## 8. Run the Environment Check Script

In the project root directory, run:

```bash
python3 scripts/check_environment.py
```

If `python3` is not available on Windows, you can also run:

```bash
python scripts/check_environment.py
```

Expected result:

- A check report is output
- You should see check results for at least Python, openpyxl, configuration file, and extension directory structure

Common errors:

- `python3` not found: Use `python` instead
- Report shows configuration file or API Key missing: Go back to step 5 to complete the setup

## 9. Recommended Final Integration Testing Order

Suggested demonstration order:

1. First run `关于` (About) to show version and environment info
2. Use `Excel 导入` (Excel Import) for forward modeling
3. Use `导出模型` (Export Model) to demonstrate reverse export
4. Use `智能对话` (Smart Chat) to demonstrate natural language modeling
5. If troubleshooting is needed, show the check report from `python3 scripts/check_environment.py`

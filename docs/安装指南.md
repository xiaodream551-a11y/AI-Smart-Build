# AI 智建安装指南

本指南面向第一次接触 Revit、pyRevit 和 Python 的用户，按顺序完成即可。

## 1. 安装 Revit

安装目标：
- 安装 `Revit 2024`

预期结果：
- Windows 开始菜单中可以找到 `Autodesk Revit 2024`
- 能正常启动 Revit

常见错误：
- 安装包不完整：重新下载安装包后再安装
- 许可证问题：先确认学校或个人账户是否有可用许可

## 2. 安装 pyRevit

安装步骤：
1. 打开 pyRevit 发布页：`https://github.com/pyrevitlabs/pyRevit/releases`
2. 下载最新版安装包
3. 双击安装，按默认选项完成

预期结果：
- 开始菜单中可以找到 `pyRevit CLI`
- 打开 Revit 后，顶部 Ribbon 中出现 `pyRevit` 选项卡

常见错误：
- Revit 中没有出现 pyRevit 选项卡：重启 Revit 再试
- 安装后仍不可用：卸载后重新安装，并确认 Revit 版本与 pyRevit 兼容

## 3. 获取项目代码

推荐方式：
1. 安装 Git
2. 打开命令提示符或 PowerShell
3. 执行：

```bash
git clone <your-repo-url> AI-Smart-Build
cd AI-Smart-Build
```

如果不会使用 Git，也可以：
1. 在仓库页面点击 `Code`
2. 选择 `Download ZIP`
3. 解压到一个你能找到的位置，例如桌面

预期结果：
- 目录中能看到 `AISmartBuild.extension`
- 目录中能看到 `scripts/install_windows.bat`

常见错误：
- 提示 `git` 不存在：先安装 Git，或改用 ZIP 下载方式
- 中文路径过深：建议把项目放到较短的英文路径，例如 `D:\Demo\AI-Smart-Build`

## 4. 运行 Windows 安装脚本

操作步骤：
1. 进入项目根目录
2. 双击运行 `scripts/install_windows.bat`

脚本会做三件事：
- 检查 Python
- 安装 `openpyxl`
- 创建 `config.json` 配置文件

预期结果：
- 命令窗口中看到 `=== 安装完成 ===`
- 用户目录下出现配置文件：

```text
C:\Users\<你的用户名>\.ai-smart-build\config.json
```

常见错误：
- 提示未检测到 Python：先安装 Python 3.10 及以上版本，并勾选“Add Python to PATH”
- `pip install openpyxl` 失败：检查网络，或改用管理员权限重新运行终端

## 5. 配置 DeepSeek API Key

操作步骤：
1. 打开配置文件：

```text
C:\Users\<你的用户名>\.ai-smart-build\config.json
```

2. 把 `DEEPSEEK_API_KEY` 改成你的真实 Key

示例：

```json
{
  "DEEPSEEK_API_KEY": "你的真实 API Key",
  "DEEPSEEK_MODEL": "deepseek-chat",
  "DEEPSEEK_API_URL": "https://api.deepseek.com/v1/chat/completions"
}
```

预期结果：
- 文件保存成功
- `DEEPSEEK_API_KEY` 不再是占位文本

常见错误：
- 忘记保存文件：修改后确认保存
- JSON 格式错误：检查最后一个字段后面不要多写逗号

## 6. 注册 pyRevit 扩展路径

操作步骤：
1. 打开 pyRevit 设置界面或 pyRevit CLI
2. 把“包含 `AISmartBuild.extension` 的目录”加入 pyRevit 自定义扩展路径

例如，如果项目在：

```text
D:\Demo\AI-Smart-Build
```

那么要加入的是：

```text
D:\Demo\AI-Smart-Build
```

不是只选 `AISmartBuild.extension` 目录本身。

预期结果：
- pyRevit 设置中能看到新增的自定义扩展路径

常见错误：
- 路径选错：必须选择“包含扩展目录的上一级目录”
- 目录被移动：如果你后来移动了项目目录，需要重新注册一次

## 7. 重载并验证插件

操作步骤：
1. 打开 Revit
2. 执行 `pyRevit Reload`
3. 检查 Ribbon 中是否出现 `AI智建` 选项卡
4. 点击 `关于` 按钮，确认版本和环境信息能显示

预期结果：
- Ribbon 中出现 `AI智建`
- 能看到一键生成、AI 对话、导出模型等按钮
- `关于` 弹窗能正常显示版本、pyRevit、Revit 和配置状态

常见错误：
- 没看到按钮：先检查扩展路径是否注册正确，再执行 `pyRevit Reload`
- 提示 API Key 未配置：回到第 5 步检查 `config.json`
- 按钮点了没反应：先运行环境检查脚本，确认依赖和目录结构正常

## 8. 运行环境检查脚本

在项目根目录执行：

```bash
python3 scripts/check_environment.py
```

如果 Windows 上没有 `python3` 命令，也可以执行：

```bash
python scripts/check_environment.py
```

预期结果：
- 输出一份中文检查报告
- 至少能看到 Python、openpyxl、配置文件、扩展目录结构的检查结果

常见错误：
- 提示 `python3` 不存在：改用 `python`
- 报告中显示配置文件或 API Key 缺失：回到第 5 步补全

## 9. 推荐的最终联调顺序

建议按下面顺序演示：
1. 先运行 `关于`，展示版本和环境
2. 用 `Excel 导入` 做正向建模
3. 用 `导出模型` 展示反向导出
4. 用 `智能对话` 展示自然语言建模
5. 如需排错，展示 `python3 scripts/check_environment.py` 的检查报告
